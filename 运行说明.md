**不要使用后台模式！** 直接在您的终端（Git Bash, PowerShell 或 cmd）中运行：


## API 接口

### 优惠券接口

- `GET /api/v1/coupons` - 获取所有优惠券
- `GET /api/v1/coupons/available` - 获取可用优惠券
- `POST /api/v1/coupons/receive` - 领取优惠券
- `POST /api/v1/coupons/calculate` - 计算最优价格

### 订单接口

- `POST /api/v1/orders` - 创建订单
- `GET /api/v1/orders` - 获取订单列表
- `GET /api/v1/orders/:id` - 获取订单详情
- `PUT /api/v1/orders/:id/pay` - 支付订单

### 秒杀接口

- `GET /api/v1/seckill/activities` - 获取秒杀活动
- `POST /api/v1/seckill/order` - 创建秒杀订单

## 优惠券叠加规则

- **满减券 + 折扣券**：可以叠加使用
- **秒杀券**：单独使用，不参与叠加

## 常见问题

### 问题：程序卡住不输出？

**解决方案**：不要使用后台模式运行，直接在终端中运行：

```bash
cd "e:/opencode测试/门票系统"
go run ./cmd/api/main.go
```

### 问题：MySQL 连接失败？

**解决方案**：
1. 检查 MySQL 服务是否在运行
2. 检查密码是否是 123456
3. 在终端中运行：`mysql -u root -p123456`

### 问题：Redis 连接失败？

**解决方案**：
1. 检查 Redis 服务是否在运行
2. 在终端中运行：`e:/Redis/redis-cli.exe ping`
3. 应该返回 `PONG`
